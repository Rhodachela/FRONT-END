# React Product Filter Table

A simple React app that displays a list of products in a table, allows filtering by product name, and optionally shows only products in stock. This project demonstrates component composition, state management with `useState`, and basic CSS styling including flexbox and table design.

## Features

- Display a list of products organized by category.
- Filter products by name using a search input.
- Option to show only products that are in stock.
- Responsive layout using flexbox.
- Hover and focus effects for better UX.
- Styled tables with scrollable wrapper for long lists.

## Technologies Used

- React
- JavaScript (ES6+)
- CSS (Flexbox, styling tables)
- Vite (for project bundling and development)
